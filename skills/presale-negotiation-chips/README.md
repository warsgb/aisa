# WPS 365 商务谈判筹码技能 (presale-negotiation-chips)

当商务谈判陷入僵局、客户要求降价时，本技能帮助售前人员生成多种"非货币筹码"交换方案，实现非零和博弈，突破价格谈判困局。

## 核心价值

- **非零和博弈思维**：跳出价格一条线死磕，寻找多方共赢方案
- **丰富筹码库**：服务、品牌、账期、产品组合四大类筹码
- **心理学原理支撑**：每个筹码都基于谈判心理学原理设计
- **角色精准匹配**：基于客户角色需求，推荐高价值筹码
- **风险边界清晰**：明确红线警示，不突破毛利底线

## 四大筹码类型

### 1. 服务类筹码

**边际成本低，客户感知价值适中**

- 削减现场人天 → 改为线上支持（节省差旅成本）
- 减少培训场次 → 改为视频教程（可复用）
- 缩短驻场周期 → 改为远程支持（降低人力成本）
- 减少定制开发 → 使用标准功能（避免项目风险）

**适用场景**：客户关注成本优化，有一定运维能力

### 2. 品牌类筹码

**客户有邀功需求时效果最佳**

- 标杆案例授权 → 签署授权书，允许宣传
- 联合新闻稿 → 共同发布合作新闻
- 高层互访 → 安排 CEO/VP 级别会面
- 行业峰会邀请 → 免费门票 + VIP 席位

**适用场景**：CEO/总经理有邀功需求，希望树立行业标杆

### 3. 账期类筹码

**现金流换价格，双赢方案**

- 预付全款折扣 → 年度一次性付款
- 长期合约优惠 → 签 3 年送 6 个月
- 分期付款方案 → 降低单期付款压力
- 季度付款优惠 → 按季度支付有折扣

**适用场景**：客户现金流紧张，或预算分期

### 4. 产品组合筹码

**结合历史已购产品，设计打包优惠**

- 老客户续费优惠 → 已购产品续费折扣
- 增购模块打包 → 新模块与老模块打包价
- 用户数扩容优惠 → 扩容用户有阶梯折扣
- 跨产品打包 → WPS 365 + 其他产品组合

**适用场景**：客户有历史采购记录，需要扩容或升级

## 快速开始

### 基础使用示例

```bash
# 场景：CEO谈判，要求降价20%
presale-negotiation-chips \
  --customer_name "某大型国企" \
  --target_role "CEO" \
  --price_gap "客户要求降20%，底线只能降10%" \
  --customer_motivation "预算锁死，但急需标杆项目邀功"
```

**输出**：推荐品牌类筹码（标杆案例授权 + 联合新闻稿），配套谈判话术

### 进阶使用示例

```bash
# 场景：采购负责人谈判，结合历史已购产品
presale-negotiation-chips \
  --customer_name "某金融机构" \
  --target_role "采购负责人" \
  --price_gap "要求降15%，底线8%" \
  --existing_products ["WPS 专业版", "金山云"] \
  --chip_types ["产品组合", "账期"]
```

**输出**：推荐产品组合筹码（老客户续费优惠 + 跨产品打包）

## 客户角色适配

| 角色 | 核心关注 | 推荐筹码 | 避免筹码 |
|------|----------|---------|---------|
| **CEO/总经理** | ROI、政绩、标杆 | 品牌类、标杆案例、高层互访 | 服务削减、技术细节 |
| **采购负责人** | 价格、性价比 | 产品组合、账期优惠 | 品牌、技术宣传 |
| **CIO/CTO** | 技术能力、运维 | 技术培训、架构咨询、服务保障 | 削减实施、品牌授权 |
| **业务负责人** | 效率、体验 | 用户扩容、培训、功能扩展 | 削减培训、服务削减 |

## 💡 谈判筹码方案

### 方案一：标杆案例授权 + 10%折扣

| 维度 | 内容 |
|------|------|
| **筹码类型** | 品牌类 |
| **具体内容** | 签署标杆案例授权书 + 联合新闻稿 + 行业峰会VIP |
| **我方成本** | 品牌背书价值（无实际成本） |
| **客户价值** | 行业标杆地位、高层邀功资本、媒体曝光度 |
| **价格让渡** | 可在原价基础上降10% |

**谈判话术**：
```
"张总，我完全理解您的预算压力。咱们换个思路：
如果您能签署标杆案例授权，允许我们将贵公司作为行业标杆案例进行宣传，
我可以在价格上给您10%的折扣。

而且咱们可以联合发布新闻稿，您也能在行业峰会上作为VIP嘉宾分享经验。
这对您向上级汇报邀功非常有帮助。您既解决了预算问题，又获得了行业标杆的品牌价值。

您看这个方案怎么样？"
```

**心理学原理**：互惠原理 - 我们给予客户标杆地位和媒体曝光，客户会感到有回报的义务

## 谈判心理学原理

本技能基于以下心理学原理设计筹码方案：

| 原理 | 机制 | 应用筹码 |
|------|------|---------|
| **互惠原理** | 给予对方某物，对方会感到有回报的义务 | 服务换折扣 |
| **社会认同** | 人们会跟随类似人群的行为 | 品牌类筹码 |
| **损失厌恶** | 人们对损失的敏感度是对同等收益的2倍 | 账期优惠 |
| **锚定效应** | 第一印象影响后续判断 | 产品组合 |
| **稀缺性原理** | 稀缺的东西更有价值 | 限量名额 |

## 红线警示

### 财务红线

- ⚠️ 任何筹码交换都不能突破公司毛利红线
- ⚠️ 价格计算需经财务确认，AI 不能代替算账
- ⚠️ 折扣权限不能超过个人授权范围

### 法律风险

- ⚠️ 品牌类筹码需法务审核授权条款
- ⚠️ Logo 使用需确认授权范围
- ⚠️ 联合新闻稿需双方审核

### 实施风险

- ⚠️ 服务类筹码需评估实施风险
- ⚠️ 削减现场支持需确认远程可行
- ⚠️ 减少培训需提供替代方案

## 最佳实践

### 1. 先分析客户动机

在使用本技能前，先明确客户的核心动机：
- 是真的预算锁死？还是想在上级面前邀功？
- 是现金流紧张？还是比价竞品？

不同动机对应不同筹码方案。

### 2. 组合使用筹码

不要只依赖一个筹码，可以组合使用：
- 品牌类 + 服务类 = 标杆授权 + 削减现场人天
- 账期类 + 产品组合 = 长期合约 + 老客户续费优惠

### 3. 话术要真诚

谈判话术要基于真实的价值交换，不要虚假承诺：
- 确实能给予的筹码才承诺
- 价格计算要经财务确认
- 品牌授权要法务审核

### 4. 留有余地

谈判是过程，不是一次到位：
- 先抛出最优方案
- 保留备选方案和底线方案
- 根据客户反应灵活调整

## 常见问题

**Q: AI 给出的价格折扣可以相信吗？**

A: 不可以。AI 只能提供筹码方案和话术，具体价格计算必须由财务确认。AI 说"可降10%"只是建议，实际数字需要财务核算后确定。

**Q: 品牌类筹码真的有效吗？**

A: 取决于客户动机。如果客户有邀功需求、希望树立行业标杆，品牌类筹码非常有效。但如果客户只是单纯关注成本，品牌类筹码效果有限。

**Q: 服务削减会影响项目成功吗？**

A: 有可能。因此服务类筹码需要评估实施风险，确认远程支持能够满足客户需求。关键节点（如上线、验收）仍建议现场支持。

**Q: 可以同时使用多个筹码吗？**

A: 可以，但要注意组合后的总成本。建议先单独评估每个筹码的成本和价值，再组合使用。
