# WPS 365 商务谈判筹码技能

商务谈判陷入僵局时，AI 帮助售前人员生成多种"非货币筹码"交换方案，跳出价格死局，实现非零和博弈。

## 核心价值

- **非零和博弈思维**：跳出价格一条线死磕，寻找多方共赢方案
- **丰富筹码库**：服务、品牌、账期、产品组合四大类筹码
- **心理学原理支撑**：每个筹码都基于谈判心理学原理设计
- **角色精准匹配**：基于客户角色需求，推荐高价值筹码
- **风险边界清晰**：明确红线警示，不突破毛利底线

## 触发条件

当用户提供以下任一信息时，启动本技能：

1. **商务谈判僵局**："客户咬死要降价20%，但我只能降10%"
2. **价格谈判困局**："谈判卡在价格上，客户不��受当前报价"
3. **寻找替代方案**："除了降价，还有什么谈判筹码可以用"
4. **设计交换条件**："客户要求XX条件，我能用什么来交换"

**示例**：
- "某国企CEO要求降20%，我底线是10%，帮我设计谈判筹码"
- "客户咬死价格，能不能用其他条件交换"
- "采购负责人要求降15%，有哪些非货币筹码可以用"

## 非零和博弈核心逻辑

### 传统价格谈判的困局

**零和博弈陷阱**：
- 在"价格"这一条线上死磕是零和博弈
- 客户降价 = 我方利润减少
- 双方立场对立，难以达成共识

### 非零和博弈思路

**扩大谈判蛋糕**：
- 寻找"边际成本低但客户感知价值高"的非货币筹码
- 用我方低成本资源，换取客户高价值感知
- 设计多方共赢的交换方案

**筹码设计原则**：
| 原则 | 说明 | 例子 |
|------|------|------|
| **边际成本低** | 我方付出的实际成本很小 | 减少现场人天（节省差旅） |
| **客户感知价值高** | 客户认为很有价值 | 培训课程、标杆案例授权 |
| **可量化价值** | 能够明确说明价值 | "节省XX万元"、"提升XX%" |
| **不突破红线** | 符合公司毛利要求 | 需财务确认具体数字 |

## 筹码类型库

本技能基于 `shared-frameworks/negotiation-chips-framework.md` 提供四大类筹码。

### 1. 服务类筹码（边际成本低，客户感知价值高）

**核心逻辑**：削减高成本服务，改用低成本替代方案

| 筹码 | 我方成本 | 客户价值 | 适用场景 |
|------|---------|---------|---------|
| 削减现场人天 → 线上支持 | 节省差旅成本 | 仍获得专业支持 | 非关键阶段 |
| 减少培训场次 → 视频教程 | 可复用内容 | 灵活学习时间 | 培训预算紧张 |
| 缩短驻场周期 → 远程支持 | 降低人力成本 | 响应更及时 | 客户有运维能力 |
| 减少定制开发 → 标准功能 | 避免项目风险 | 系统更稳定 | 需求可标准化 |

### 2. 品牌类筹码（客户有邀功需求时有效）

**核心逻辑**：用品牌曝光、标杆案例价值交换价格

| 筹码 | 我方成本 | 客户价值 | 适用场景 |
|------|---------|---------|---------|
| 标杆案例授权 | 品牌背书 | 行业标杆地位 | CEO/总经理邀功 |
| 联合新闻稿 | 营销资源 | 媒体曝光度 | 需要宣传项目 |
| 高层互访 | 高层时间 | 提升合作层级 | 战略合作项目 |
| 行业峰会邀请 | 门票+VIP席位 | 行业影响力 | 业务负责人 |

### 3. 账期类筹码（现金流换价格）

**核心逻辑**：用优惠现金流条件，换取价格让渡

| 筹码 | 我方收益 | 客户价值 | 适用场景 |
|------|---------|---------|---------|
| 预付全款折扣 | 现金流改善 | X%折扣 | 客户现金充裕 |
| 长期合约优惠 | 锁定长期收入 | 签3年送6个月 | 长期合作意向 |
| 分期付款方案 | 可能收利息 | 降低单期压力 | 预算分期 |
| 季度付款优惠 | 现金流稳定 | 按季度支付 | 财务规范 |

### 4. 产品组合筹码（结合历史已购产品）

**核心逻辑**：结合历史已购产品，设计打包优惠方案

| 筹码 | 我方成本 | 客户价值 | 适用场景 |
|------|---------|---------|---------|
| 老客户续费优惠 | 锁定续费 | 已购产品续费折扣 | 续费谈判 |
| 增购模块打包 | 提升客单价 | 新老模块打包价 | 功能扩容 |
| 用户数扩容优惠 | 规模效应 | 阶梯折扣 | 用户数增长 |
| 跨产品打包 | 生态协同 | WPS 365 + 其他产品 | 多产品需求 |

## 谈判话术框架

### 标准话术结构

每个筹码配套专业谈判话术，按以下结构展开：

1. **认可**：认可客户的预算压力/价格要求
2. **转折**：提出替代方案，扩大谈判蛋糕
3. **价值量化**：说明客户获得的价值
4. **行动呼吁**：引导客户接受方案

### 话术模板

**开场认可话术**：
- "我理解您的预算压力..."
- "您的要求我听到，咱们一起想想办法..."
- "价格确实是重要因素，咱们看看有没有其他方案..."

**转折提议话术**：
- "如果您能在XX方面支持我们，我们可以在价格上..."
- "咱们换个思路，您看这样行不行..."
- "我有个提议，您看看能不能接受..."

**价值量化话术**：
- "这样您既解决了预算问题，又获得了XX价值..."
- "这个方案对您来说，总价值是..."
- "从TCO角度看，您实际上节省了..."

**行动呼吁话术**：
- "您看这个方案怎么样？"
- "咱们先从这个方案开始，细节再细化..."
- "如果您同意这个方向，我马上准备方案..."

## 心理学原理库

每个筹码背后都有心理学原理支撑，增强谈判说服力。

### 核心心理学原理

| 原理 | 机制 | 应用筹码 | 话术示例 |
|------|------|---------|---------|
| **互惠原理** | 给予对方某物，对方会感到有回报的义务 | 服务换折扣 | "我们削减现场人天为您节省成本，也希望您在价格上支持我们" |
| **社会认同** | 人们会跟随类似人群的行为 | 品牌类筹码 | "XX行业标杆企业也是这样合作的" |
| **损失厌恶** | 人们对损失的敏感度是对同等收益的2倍 | 账期优惠 | "限时优惠，错过就损失了X万元" |
| **锚定效应** | 第一印象影响后续判断 | 产品组合 | "原价XX万，打包价XX万，您节省了XX万" |
| **稀缺性原理** | 稀缺的东西更有价值 | 限量名额 | "这个优惠只有3个名额，先到先得" |

### 话术应用示例

**互惠原理应用**：
```
"我理解您的预算压力。我们愿意削减30%的现场人天，改为远程支持，
这为您节省了XX万元成本。作为交换，也希望您在价格上能支持我们的底线。
这样您既获得了解决方案，又节省了成本，咱们双赢。"
```

**锚定效应应用**：
```
"如果我们单独计算，培训服务价值XX万，实施服务价值XX万，
现在打包在一起，总价XX万，您相当于节��了XX万。"
```

**损失厌恶应用**：
```
"这个长期合约优惠只有本周有效，如果您今天签约，
可以享受签3年送6个月的优惠。如果错过，就损失了6个月的价值。"
```

## 红线警示

### 底线思维

**核心红线**：
- ⚠️ 任何筹码交换都不能突破公司毛利红线
- ⚠️ 价格计算需经财务确认，AI 不能代替算账
- ⚠️ 品牌类筹码需法务审核授权条款
- ⚠️ 服务类筹码需评估实施风险

### 财务确认清单

在提供任何价格相关方案前，必须确认：

- [ ] **毛利红线**：方案不突破公司最低毛利率要求
- [ ] **现金流影响**：账期方案需财务评估现金流影响
- [ ] **成本核算**：服务削减需评估实际成本节省
- [ ] **折扣权限**：确认折扣在个人授权范围内

### 法律风险提示

品牌类筹码注意事项：
- [ ] **授权条款**：案例授权书需法务审核
- [ ] **Logo使用**：确认客户Logo使用授权范围
- [ ] **新闻稿内容**：联合新闻稿需双方审核
- [ ] **隐私保护**：注意客户敏感信息保护

### 实施风险提示

服务类筹码注意事项：
- [ ] **实施风险**：削减现场支持需评估实施成功率
- [ ] **培训效果**：减少培训场次需提供替代学习方案
- [ ] **定制开发**：减少定制需确认需求可用标准功能满足
- [ ] **运维责任**：缩短驻场需明确运维责任边界
