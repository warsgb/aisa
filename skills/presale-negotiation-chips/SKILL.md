## 质量检查清单

在生成谈判筹码方案时，AI 需确保：

- [ ] **角色匹配**：筹码方案与客户角色关注点高度匹配
- [ ] **动机洞察**：筹码设计与客户核心动机一致
- [ ] **成本合理**：我方成本评估准确，边际成本低
- [ ] **价值明确**：客户价值可量化，感知价值高
- [ ] **话术专业**：谈判话术遵循认可-转折-价值-行动框架
- [ ] **心理学支撑**：每个筹码都有心理学原理支撑
- [ ] **红线清晰**：明确提示财务、法律、实施风险
- [ ] **数据准确**：引用的TCO、效率数据来自框架文件

## 💡 谈判筹码方案

### 方案一：标杆案例授权 + 10%折扣

| 维度 | 内容 |
|------|------|
| **筹码类型** | 品牌类 |
| **具体内容** | 客户签署标杆案例授权书，允许WPS使用企业名称和Logo进行宣传；配合拍摄案例宣传片、参加行业峰会 |
| **我方成本** | 品牌背书价值（无实际成本） |
| **客户价值** | 行业标杆地位、高层邀功资本、媒体曝光度 |
| **价格让渡** | 可在原价基础上降10% |
| **适用场景** | 客户有邀功需求、希望树立行业标杆形象 |

**谈判话术**：
```
[认可] 张总，我完全理解您的预算压力。年度预算锁死确实是个难题。

[转折] 咱们换个思路，您看这样行不行：如果您能签署标杆案例授权，
允许我们将贵公司作为行业标杆案例进行宣传，我可以在价格上给您10%的折扣。

[价值] 这样您既解决了预算问题，又获得了行业标杆的品牌价值。
咱们可以联合发布新闻稿，您也能在行业峰会上作为VIP嘉宾分享经验。
这对您向上级汇报邀功非常有帮助。

[行动] 您看这个方案怎么样？如果同意，我马上准备授权书和新闻稿草案。
```

**心理学原理**：互惠原理 - 我们给予客户标杆地位和媒体曝光，客户会感到有回报的义务，在价格上更易达成一致

**风险提示**：
- 需法务审核授权书条款
- 确认客户Logo使用授权范围
- 新闻稿内容需双方审核

---

### 方案二：削减现场人天 + 8%折扣

| 维度 | 内容 |
|------|------|
| **筹码类型** | 服务类 |
| **具体内容** | 原计划20人天现场支持，削减为10人天，其余改为远程支持 |
| **我方成本** | 节省差旅成本约X万元 |
| **客户价值** | 节省差旅成本X万元，获得同样专业的支持 |
| **价格让渡** | 可在原价基础上降8% |
| **适用场景** | 客户有一定运维能力，非关键阶段 |

**谈判话术**：
```
[认可] 张总，预算问题我理解。

[转折] 咱们在服务交付方式上调整一下如何？
原计划20人天现场支持，我们可以削减为10人天，其余改为远程专家支持。
这样既为您节省了差旅成本，我们也能在价格上给您8%的折扣。

[价值] 远程支持其实响应更及时，而且我们的专家团队随时待命。
从成本角度看，您既节省了差旅费，又获得了价格优惠，总体更划算。

[行动] 您觉得这个方案怎么样？
```

**心理学原理**：锚定效应 - 原价20人天作为锚点，10人天+折扣显得更优惠

**风险提示**：
- 需评估远程支持是否能满足实施需求
- 明确远程支持的响应时间和SLA
- 关键节点（如上线）仍需现场支持

---

### 方案三：长期合约 + 12%折扣

| 维度 | 内容 |
|------|------|
| **筹码类型** | 账期类+产品组合 |
| **具体内容** | 签订3年合约，赠送6个月服务；年度一次性付款享受额外2%折扣 |
| **我方成本** | 锁定长期收入，改善现金流 |
| **客户价值** | 3.5年的价格享受3年服务，长期锁定价格 |
| **价格让渡** | 可在原价基础上降12% |
| **适用场景** | 客户有长期使用意愿，预算可分期 |

**谈判话术**：
```
[认可] 张总，预算确实是个问题。

[转折] 如果咱们能把合作周期拉长，我就能给您更大的优惠。
签3年送6个月，相当于您用3年的预算享受3.5年的服务。
而且如果年度一次性付款，还能再享受2%的折扣，总共能降12%。

[价值] 这样既解决了您的年度预算问题，又锁定了长期价格，
避免未来涨价风险。对您来说，总价值提升了17%。

[行动] 您觉得长期合作这个方向怎么样？
```

**心理学原理**：损失厌恶 + 稀缺性 - "限时优惠"营造紧迫感，客户害怕错过优惠

**风险提示**：
- 需财务确认3年合约的价格计算
- 评估长期合约的价格锁定风险
- 确认客户有3年预算承诺
