# 病毒标题生成器 (viral-title-generator)

生成吸引眼球的爆款文案标题，适用于文章、视频、社交媒体帖子、广告文案等。支持多种风格：悬念式、数字式、提问式、情感共鸣式、权威背书式等。

## 快速开始

**触发条件**：
- 用户需要为文章、视频、社交媒体帖子生成标题
- 用户需要提高内容点击率和传播性
- 用户需要为不同平台（公众号、抖音、小红书、知乎、B站）定制标题

**使用示例**：
```
帮我写一个关于"时间管理"的爆款标题
生成适合抖音的职场干货标题
为我的减肥文章写5个吸引人的标题
```

## 核心功能

1. **数字具体化**：具体数字 + 核心内容 + 利益点
   - "30天学会Python，从入门到实战"
   - "这3招，让你效率提升200%"

2. **悬念反差型**：违背常识/制造冲突 + 留悬念
   - "年薪百万的我，每天却只吃一顿饭"
   - "这个被所有人嘲笑的决定，改变了我的一生"

3. **情感共鸣型**：戳中痛点/唤起情绪 + 解决方案
   - "成年人的崩溃，从一张体检报告开始"
   - "30岁还单身，我真的错了吗？"

4. **权威背书型**：名人/数据/机构 + 核心观点
   - "哈佛研究：这5个习惯决定你90%的人生"
   - "马斯克每天必做的3件事"

5. **提问互动型**：问题 + 利益/悬念
   - "为什么越努力的人越穷？真相扎心了"
   - "你知道月薪2万和月薪20万的区别吗？"

## 平台适配建议

| 平台 | 标题特点 | 字数建议 | 风格偏好 |
|------|----------|----------|----------|
| 公众号 | 深度、共鸣 | 15-25字 | 走心、观点 |
| 抖音 | 短平快、冲击 | 8-15字 | 夸张、反转 |
| 小红书 | 实用、种草 | 12-20字 | 亲切、干货 |
| 知乎 | 专业、好奇 | 15-30字 | 理性、深度 |
| B站 | 趣味、年轻化 | 10-20字 | 玩梗、创意 |

## 输出示例

```
目标平台：公众号
内容风格：干货教程

生成标题：
1. 《每天多出3小时！这5个时间管理技巧让我效率翻倍》
2. 《为什么你越忙越穷？99%的人都在假努力》
3. 《我从"拖延癌"到"效率控"，只做对了这一件事》
4. 《哈佛商学院推荐：顶级精英都在用的3个时间法则》
5. 《别再做"时间穷人"！这份指南让你掌控人生》
```

## 标题优化检查清单

生成标题后，用以下标准检验：
- [ ] 是否包含具体数字或具体利益点？
- [ ] 是否制造了好奇或情感共鸣？
- [ ] 是否避免了标题党（与内容严重不符）？
- [ ] 是否适合目标平台和受众？
- [ ] 字数是否在合理范围内？
- [ ] 是否有错别字或语病？
