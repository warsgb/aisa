# 行业黑话速成 (presale-industry-jargon)

快速掌握行业术语，建立信任，避免"露怯"。提供行业四级分类体系，黑话三维表结构，售前场景应用。

## 快速开始

**触发条件**：
- 用户需要快速了解行业术语
- 用户准备不足，需要速成

**使用示例**：
```
银行行业的常用术语有哪些？
保险业黑话速成
```

## 核心功能

1. **行业四级分类**：行业/子行业/场景/角色
2. **黑话三维表**：黑话/一句话解释/售前怎么用/风险提示
3. **装懂红线**：避免露怯的风险提示
4. **万能句式**："这块我不太熟，能不能请教您一下..."

## 输出示例

```
黑话三维表：
| 黑话 | 一句话懂它 | 售前怎么用 | 风险提示 |
|------|-----------|-----------|----------|
| AUM | 客户在我们这存了多少钱 | "我们的方案可以帮助提升AUM" | 不懂别乱用 |
```
