# WPS 365 高层拜访故事线规划

> 生成针对 CXO 级别高层的拜访故事线思路，将技术痛点升华为商业指标

## 简介

本技能帮助售前人员快速构建打动 CXO 级别高层的拜访故��线思路。基于战略解码分析结果，将技术痛点升华为 CXO 关注的商业指标，提供清晰的"北极星方案"和结构化的 SCQA 框架。

## 核心特性

- ✅ **技术痛点升华为商业指标**：将"系统分散"转化为"效率提升30%，打造行业标杆"

- ✅ **SCQA 结构完整**：背景、冲突、问题、方案，逻辑清晰递进

- ✅ **三种使用模式**：自动分析、已有结果、快速模板

## 高层四大关注点

| 关注点 | 核心诉求 | WPS 365 对接话术 |
|--------|----------|------------------|
| **影响力** | 个人政绩、打造标杆、行业领先 | "打造成行业数字化标杆" |
| **业务增长** | 核心指标、资产增长、业务创新 | "效率提升→业务响应快→收入增长" |

| **提效** | 世界500强实践、同行对比 | "某同行效率提升50%" |

## 故事线模型

### SCQA（传统型）

**结构**：Situation（背景）→ Complication（冲突）→ Question（问题）→ Answer（方案）

**适用场景**：
- 初次拜访
- 保守型高层
- 需要铺垫背景

### QSCA（冲击型）

**结构**：Question（问题）→ Situation（背景）→ Complication（冲突）→ Answer（方案）

**适用场景**：
- 时间紧张（15分钟内）
- 激进型高层
- 需要快速抓住注意力

## 输出格式

```
1. 北极星方案（一句话）
2. SCQA 故事线框架
   - Situation 背景（2-3段）
   - Complication 冲突（2-3段）
   - Question 问题（1-2个）
   - Answer 方案（5-7个论据）
3. 关键数据支撑
4. 核心话术参考
5. 后续行动建议
```
