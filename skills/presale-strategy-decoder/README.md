# 财报年报战略解码 (presale-strategy-decoder)

从客户年报、战略规划文件中解码销售机会，自动识别战略关键词，深度推理业务痛点，精准匹配 WPS 365 解决方案。

## 快速开始

**触发条件**：
- 用户提供企业年报、战略规划文件、官网战略信息
- 用户提供公司名称（自动搜索年报）

**使用示例**：
```
帮我分析招商银行2025年年报
解码国家电网的战略规划
```

## 核心功能

1. **关键词提取与分类**：降本增效/合规风控/数字创新/安全可信
2. **痛点深度推理**：基于关键词推理业务痛点
3. **WPS 365 匹配**：精准匹配解决方案
4. **生成三种话术**：初次见面/方案汇报/投标答辩
