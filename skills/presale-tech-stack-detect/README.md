# 技术栈侦探 (presale-tech-stack-detect)

从客户 JD（职位描述）反推技术架构，识别技术债务信号，分析 WPS 365 部署兼容性，生成技术对接话术。

## 快速开始

**触发条件**：
- 用户提供 JD 文件、JD 文本、招聘链接
- 用户提供公司名称（自动搜索 JD）

**使用示例**：
```
帮我分析这个JD：[JD文本]
从招商银行的JD推断技术栈
```

## 核心功能

1. **技术栈识别**：编程语言/数据库/中间件/认证/部署
2. **技术债务信号**：重构/性能问题/集成痛点/信创改造
3. **部署兼容性分析**：公有云/私有云/混合云/信创版
4. **生成技术对接话术**：初次接触/技术对接/信创改造

## 输出示例

```
技术栈分析表：
| 技术分类 | 技术栈 | 版本要求 | WPS 365 兼容性 | 集成方案 |

技术债务信号表：
| 信号类型 | JD 原文 | 技术债务推测 | WPS 365 机会 | 优先级 |
```
