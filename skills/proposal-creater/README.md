# WPS 365 标书响应文档生成助手 (Proposal Creator)

根据招标文件自动生成专业的 WPS 365 技术响应文档。AI 智能分析招标类型，精准匹配产品能力，结合行业最佳实践，快速输出投标文档。

---

## 核心价值

| 价值 | 说明 |
|------|------|
| **AI 智能分析** | 自动识别招标类型（办公协同/文档中台/视频会议/信创改造/数字化转型/安全合规） |
| **精准需求匹配** | 基于招标文件逐条响应，确保无遗漏 |
| **行业最佳实践** | 结合行业痛点和标杆案例，提升说服力 |
| **专业文档结构** | 符合投标文档规范，包含技术方案、实施计划、服务保障 |
| **高效输出** | 快速生成专业文档，提升售前效率 |

---

## 快速开始

### 基本用法

```
用户：帮我生成某省国资委办公协同项目的投标文件，客户是某省国资委，行业是政府

系统执行：
1. 分析招标文件 → 识别招标类型（办公协同 + 信创改造）
2. 提取核心需求 → 统一入口、流程审批、信创适配、安全合规
3. 匹配 WPS 365 能力 → 协同文档、金山协作、私有化部署、七重权限
4. 生成技术响应文档 → 项目理解 + 技术方案 + 实施计划 + 服务保障
```

### 参数说明

| 参数 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| tender_content | text | 是 | 招标文件内容（完整招标文件或技术要求部分） | [招标文件文本] |
| customer_industry | string | 是 | 客户行业 | "政府"、"金融"、"能源" |
| customer_name | string | 否 | 客户名称 | "某省国资委" |
| tender_type | string | 否 | 招标类型（AI自动分析） | "办公协同" |
| reference_project | string | 否 | 参考项目案例 | "湖北文旅" |
| response_volume | string | 否 | 响应篇幅 | "concise"/"standard"/"detailed" |
| include_pricing | boolean | 否 | 是否包含报价方案 | false |

---

## AI 招标类型自动识别

基于招标文件内容自动判断招标类型：

| 招标类型 | 关键词识别 | 典型需求 |
|---------|-----------|----------|
| 办公协同 | 协同办公、即时通讯、审批流程、组织架构 | 统一入口、多端同步、流程审批 |
| 文档中台 | 文档管理、知识库、内容管理、版本控制 | 集中存储、权限管控、知识沉淀 |
| 视频会议 | 视频会议、会场建设、远程沟通、应急指挥 | 高清会议、会议管理、录播点播 |
| 信创改造 | 国产化、信创、适配、替代、自主可控 | 信创全栈、格式兼容、私有化部署 |
| 数字化转型 | 数字化转型、数据中台、业务数字化 | AI 赋能、数据整合、智能办公 |
| 安全合规 | 安全、等保、审计、权限、合规 | 七重权限、审计留痕、数据安全 |

## 输出文档结构

### 标准篇幅 (standard)

```
# 技术响应文件

## 一、项目理解
  - 1.1 需求分析
  - 1.2 重点难点理解

## 二、总体技术方案
  - 2.1 方案设计原则
  - 2.2 架构设计
  - 2.3 产品选型

## 三、详细技术方案
  - 3.1 [模块1] 方案
  - 3.2 [模块2] 方案

## 四、项目实施计划
  - 4.1 实施原则
  - 4.2 里程碑计划

## 五、服务保障体系
  - 5.1 服务范围
  - 5.2 响应承诺

## 六、案例与资质
  - 6.1 标杆案例
  - 6.2 资质证书
```

### 篇幅级别

| 级别 | 说明 | 适用场景 |
|------|------|----------|
| concise | 简短版 | 快速响应、时间紧迫 |
| standard | 标准版 | 常规投标、推荐使用 |
| detailed | 详细版 | 大型项目、复杂需求 |

## 质量检查清单

使用本技能生成的文档后，请进行以下质量检查：

- [ ] **招标需求完整响应**：招标文件中的每项需求都有对应响应
- [ ] **招标类型识别准确**：AI 自动识别的招标类型合理
- [ ] **WPS 365 能力匹配**：产品能力与招标需求准确对应
- [ ] **行业案例适当引用**：标杆案例与客户行业匹配
- [ ] **文档结构完整**：包含技术方案、实施计划、服务保障
- [ ] **篇幅符合要求**：输出长度符合 response_volume 参数设定
- [ ] **专业表达规范**：术语统一、文字专业、结构清晰
- [ ] **差异化价值突出**：突出 WPS 365 相对竞品的优势
