# 关键摘要洞察模板

**说明**：本模板用于在报告开头为销售/售前团队提供关键洞察，帮助快速了解客户价值和机会。

**使用场景**：
- 位于报告开头，在"执行摘要"之前
- 为销售/售前团队提供5分钟快速决策信息
- 不包含行动计划、销售策略等实施细节
- 只包含核心洞察和事实

## 使用指南

### 必填项（核心8点）

1. ✅ **机会评估**：市场规模、紧迫性、机会评级
2. ✅ **现有合作基础**（或市场切入点）
3. ✅ **关键决策人观点**：至少3个核心观点，标注与WPS 365匹配度
4. ✅ **客户痛点**：至少3个痛点，标注WPS 365匹配度
5. ✅ **竞争优势**：对比主要竞争对手
6. ✅ **次要决策人/细分场景**（可选，视情况）
7. ✅ **子公司机会**（可选，视情况）
8. ✅ **关键风险**：至少3个风险及应对

### 评分标准（机会评级）

| 评分 | 说明 |
|------|------|
| ⭐⭐⭐⭐⭐ 五星级 | 市场规模大（IT投入50亿+）、紧迫性高（项目正在进行）、决策人观点契合、有合作基础 |
| ⭐⭐⭐⭐ 四星级 | 市场规模大（IT投入10-50亿）、或紧迫性高、或决策人观点契合 |
| ⭐⭐⭐ 三星级 | 市场规模中等（IT投入1-10亿），需要评估机会 |
| ⭐⭐ 二星级 | 市场规模小（IT投入<1亿），或低优先级 |

### 符号说明

- 🔥 **火号**：表示紧急或高优先级
- ✅ **勾号**：表示正面优势或机会
- ⚠️ **警告号**：表示风险或需要注意的事项
- ⭐ **星号**：表示评分或匹配程度
- 🎯 **靶心**：表示关键点或精准匹配
- 💡 **灯泡**：表示洞察或关键发现
- 🏆 **奖杯**：表示竞争优势
- 💰 **钱袋**：表示商业价值或收益
- 🎪 **帐篷**：表示细分场景或次要决策人

### 质量检查清单

完成关键摘要洞察后，进行以下检查：

- [ ] **简洁明了**：每点1-3句话，不超过半页
- [ ] **数据支撑**：每个观点都有数据或原文引用支撑
- [ ] **战略高度**：聚焦战略价值，不陷入细节
- [ ] **销售导向**：帮助销售快速判断是否跟进
- [ ] **可信度高**：来源明确，不夸大
- [ ] **可执行性**：洞察能够指导下一步行动
