## 数据来源说明

### 中国政府采购网（主要数据源）

**网址**：http://www.ccgp.gov.cn

**适用对象**：所有使用政府采购资金的金融机构（国有银行、政策性银行、地方金融平台等）

**搜索方法**：

1. **基本搜索**：
   - URL格式：`http://www.ccgp.gov.cn/search?keywords=公司名称`
   - 示例：`http://www.ccgp.gov.cn/search?keywords=招商银行`

2. **按时间筛选**：
   - 在搜索结果页选择时间范围（最近1周/1月/3月/半年/1年）
   - 优先查看最近1月的招标公告

3. **按地区筛选**：
   - 选择公司总部所在省份或业务覆盖地区
   - 示例：招商银行重点看深圳市、广东省采购网

**使用WebReader抓取配置**：
```json
{
  "url": "http://www.ccgp.gov.cn/search?keywords=招商银行",
  "return_format": "markdown",
  "retain_images": false,
  "timeout": 30,
  "with_links_summary": true
}
```

**关键信息提取**：
- 项目名称：从标题提取
- 采购单位：从公告正文提取
- 预算金额：查找"预算金额"、"采购预算"等关键词
- 截止日期：查找"投标截止时间"、"递交截止时间"等关键词
- 联系方式：查找"联系人"、"联系电话"等关键词

## WPS 365 匹配度评估

### 高匹配度特征

| 特征 | 说明 | 示例关键词 |
|------|------|----------|
| 协同办公需求 | 明确提出协同办公、办公系统 | 协同办公、办公系统、办公软件 |
| 文档管理需求 | 明确提出文档管理、知识管理 | 文档管理、知识管理、档案管理 |
| 国产化要求 | 明确提出国产化、信创要求 | 国产化、信创、自主可控、国产软件 |
| 预算范围匹配 | 预算在100万-1000万之间 | 预算金额、采购预算 |

### 中匹配度特征

| 特征 | 说明 | 示例关键词 |
|------|------|----------|
| 部分功能需求 | 提到部分WPS 365功能 | 会议室管理、审批流程、即时通讯 |
| 预算偏低 | 预算在50万-100万之间 | 预算金额、采购预算 |
| 需求不明确 | 需求描述模糊 | 综合办公系统、信息化建设 |

### 低匹配度特征

| 特征 | 说明 | 示例关键词 |
|------|------|----------|
| 完全不相关 | 核心需求与WPS 365无关 | 核心业务系统、交易系统、风控系统 |
| 预算过高 | 预算超过5000万 | 大型系统集成、整体IT外包 |
| 预算过低 | 预算低于50万 | 单一功能模块、小工具采购 |

## 跟进建议生成

### 即将截止项目（截止日期<7天）

**跟进策略**：紧急跟进
**具体动作**：
1. 立即联系采购方，确认招标要求
2. 评估是否具备投标资格
3. 如具备资格，立即组建投标团队
4. 准备技术方案和商务报价

**WPS 365价值主张**：
- 强调快速交付能力（标准产品，1个月内上线）
- 强调金融行业案例（同类型客户案例）
- 强调信创能力（全栈国产化）

### 新发布项目（公告日期<3天）

**跟进策略**：早期介入
**具体动作**：
1. 分析招标需求，评估匹配度
2. 联系采购方，了解项目背景
3. 提供产品介绍和案例资料
4. 建立初步联系，为后续投标铺垫

**WPS 365价值主张**：
- 提供产品演示和试用
- 安排技术交流会
- 提供定制化方案建议

### 预算充足项目（预算>500万）

**跟进策略**：重点跟进
**具体动作**：
1. 深度分析需求，定制化方案
2. 提供POC验证支持
3. 安排高层对接
4. 准备详细的技术方案和商务方案

**WPS 365价值主张**：
- 强调企业完整版能力
- 强调私有化部署能力
- 强调定制开发能力

### 预算紧张项目（预算<100万）

**跟进策略**：标准跟进
**具体动作**：
1. 推荐标准版产品
2. 提供标准报价
3. 简化方案，突出核心价值

**WPS 365价值主张**：
- 强调性价比
- 强调快速上线
- 强调标准化产品

## 关键洞察提取

### 1. 即将截止项目识别

**识别方法**：查找投标截止日期在未来7天内的项目

**价值**：
- 竞争可能较少（部分供应商来不及准备）
- 可体现快速响应能力

**洞察模板**：
> 发现X个即将截止的招标项目（最近截止：[项目名称]，距截止X天），建议紧急跟进。

### 2. 历史中标趋势分析

**分析方法**：统计最近1年的中标项目，分析采购节奏

**价值**：
- 了解客户采购习惯（季度末/年末采购高峰）
- 预测未来采购机会

**洞察模板**：
> 最近1年共发现X个招标项目，其中Q1/Q2/Q3/Q4分别X个，客户采购集中在Q4，建议提前准备。

### 3. 预算规模分析

**分析方法**：统计最近1年的中标项目预算，分析预算范围

**价值**：
- 了解客户采购预算水平
- 判断是否为目标客户

**洞察模板**：
> 最近1年招标项目预算范围在X万-Y万，平均预算Z万，与WPS 365目标客户匹配度高。

### 4. 采购内容偏好分析

**分析方法**：分析历史招标项目的采购内容，总结偏好

**价值**：
- 了解客户关注的功能点
- 优化产品介绍和方案设计

**洞察模板**：
> 客户历史采购偏好：协同办公、文档管理、移动办公，与WPS 365核心能力高度匹配。

## 数据时效性管理

### 更新频率建议

| 数据类型 | 更新频率 | 说明 |
|----------|----------|------|
| 招投标信息 | 实时 | 每次查询时获取最新数据 |
| 项目状态 | 每日 | 跟踪已关注项目的状态变化 |
| 历史趋势 | 每月 | 每月更新一次趋势分析 |

### 数据时效性标注

在报告中增加时效性可视化标识：
- 7天内：🟢 最新
- 7-30天：🟡 较新
- 30天以上：🔴 需更新

**标注示例**：
```markdown
**数据时效性**：🟢 最新（公告日期8天前）
```

## 触发条件

满足以下任一条件即触发招投标信息查询：

1. **用户明确要求**：
   - "帮我查一下XX银行的招投标信息"
   - "XX银行最近有什么招标项目"

2. **research_depth 为 comprehensive**：
   - 自动查询最近6个月招投标信息

3. **IT投入>1亿/年**：
   - 大客户，值得深入挖掘招投标机会

4. **include_bidding 参数为 true**：
   - 用户明确启用招投标信息查询
