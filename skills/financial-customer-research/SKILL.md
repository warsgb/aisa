# WPS 365 金融行业与国企客户研究技能

为 WPS 365 售前团队提供专业的金融行业与国企客户研究支持，自动识别客户类型，生成结构��分析报告和解决方案匹配。

## 客户类型自动识别

### 金融行业识别关键词

**银行业**：银行、农商行、城商行、农信社、村镇银行
**保险业**：保险、寿险、财险、再保险、保险集团
**证券业**：证券、投行、基金、信托、期货、资管
**其他金融**：金融、金控、租赁、保理、担保、小贷

### 国企识别关键词

**央企**：国家、中国、中央、集团、总公司、有限公司（央企特有）
**地方国企**：省属、市属、国资、控股、投资、集团、建设、交通、能源
**特定行业**：电网、电力、石油、石化、铁路、航空、通信、邮政、烟草、盐业
**事业单位**：大学、医院、研究院、所、中心、台、馆

### 混合型识别

**金融国企**：国有银行、政策性银行、国有保险、国有证券
**产业金融**：财务公司、汽车金融、消费金融（通常隶属大型国企集团）

## 触发条件

当用户提供以下任一信息时，启动本技能：

### 金融行业客户
- **公司名称**：如"招商银行"、"中国平安"、"中信证券"
- **公司简称**：如"招行"、"平安"
- **部门名称**：如"零售金融部"、"寿险事业部"、"投行部"
- **岗位/角色**：如"CIO"、"IT总监"、"数字化转型负责人"
- **行业关键词**：如"银行业"、"保险业"、"证券业"

### 国企客户
- **公司名称**：如"国家电网"、"中石油"、"中国移动"、"国家能源集团"
- **公司简称**：如"国网"、"中石油"
- **部门名称**：如"办公厅"、"信息化部"、"运营管理部"
- **岗位/角色**：如"CIO"、"信息中心主任"、"科技部总经理"
- **行业关键词**：如"央企"、"国企"、"能源"、"电力"、"交通"

## 使用流程

### ⚠️ 重要：年份使用规则

**当前年份规则**：
- 报告生成时自动使用**当前系统年份**（如2026年）
- 搜索年报时优先搜索**最近年度年报**（如当前2026年，则搜索2025年年报）
- 搜索关键词格式：`公司名称 + [当前年份-1] + 年报`（如"天津银行 2025 年报"）
- 报告中所有日期标注使用当前年份

**示例**：
```
���前时间：2026年2月
搜索年报：2025年年报、2025年年度报告
报告日期：2026-02-10
数据时效：🟢 最新（2025年年报数据）
```

### 第一步：明确研究目标

确认用户输入的信息类型：

1. **完整信息**：公司 + 部门 + 岗位
   - 示例："招商银行零售金融部IT总监"

2. **部分信息**：仅公司或仅部门
   - 示例："招商银行" 或 "零售金融部"

3. **行业研究**：仅行业类型
   - 示例："银行业数字化转型"

### 第二步：数据获取（按优先级）

按以下优先级获取客户信息：

#### 优先级 1：用户上传文件（最优先）
- 检查用户是否上传了客户资料、招标文件、内部文档
- 使用 Read 工具读取文件内��
- 提取关键信息：公司规模、业务结构、痛点、预算等

#### 优先级 2：权威公告来源（根据客户类型选择）

##### 2A. 金融行业客户：巨潮网公告（权威来源）
- URL 格式：`http://www.cninfo.com.cn/new/search/keyword?searchkey=公司全称`
- 重点获取：
  - 年度报告（了解业务规模、战略方向）
  - 半年度报告（了解最新业务进展）
  - 重大事项公告（了解IT项目、数字化转型）
- 使用 WebReader 工具配置：
  ```json
  {
    "url": "http://www.cninfo.com.cn/new/search/keyword?searchkey=招商银行",
    "return_format": "markdown",
    "retain_images": false,
    "timeout": 30,
    "with_links_summary": true
  }
  ```

##### 2B. 国企客户：国资委及政府平台（权威来源）

**央企数据源**：

1. **国务院国资委官网**（最权威）

     - "人事任免" - 了解高层变动

2. **央企官网投资者关系**
   - 社会责任报告（了解战略方向、IT投入）
   - 年度报告（了解业务规模、财务状况）
   - 重大事项公告（了解IT项目、数字化采购）

3. **中国政府采购网**
   - URL：http://www.ccgp.gov.cn

#### 优先级 6：实时信息查询（可选触发）

**6.1 招投标信息查询**

**触发条件**（满足任一即触发）：
1. 用户明确要求："帮我查一下XX银行的招投标信息"
2. `research_depth` 为 `comprehensive`：自动查询最近6个月招投标信息
3. IT投入>1亿/年：大客户，值得深入挖掘
4. `include_bidding` 参数为 `true`

**数据获取方法**（按优先级，全面搜索）：

1. **中国政府采购网**（免费，主要数据源）
   - URL格式：`http://www.ccgp.gov.cn/search?keywords=公司名称`
   - WebReader配置示例：
   ```json
   {
     "url": "http://www.ccgp.gov.cn/search?keywords=招商银行",
     "return_format": "markdown",
     "retain_images": false,
     "timeout": 30,
     "with_links_summary": true
   }
   ```

2. **各省市政府采购网**（免费，补充数据源）
   - 根据客户总部所在地及业务覆盖区域选择
   - 北京：http://www.ccgp-beijing.gov.cn
   - 上海：http://www.zfcg.sh.gov.cn
   - 深圳：http://www.zfcg.sz.gov.cn
   - 广东：http://www.gdgpo.czt.gd.gov.cn
   - 天津：http://www.ccgp-tianjin.gov.cn
   - 搜索关键词：`公司名称`、`协同办公`、`办公系统`、`OA系统`

3. **公共资源交易平台**（免费，重要数据源）
   - 全国公共资源交易平台：http://www.ggzy.gov.cn
   - 搜索关键词：`公司名称 + 招标`、`公司名称 + 采购`
   - 关注：IT系统采购、软件采购、服务采购

4. **WebSearch 多维度搜索**（免费，覆盖面广）
   - 搜索组合1：`公司名称 + 招投标 + [当前年份]`
   - 搜索组合2：`公司名称 + 协同办公 + 招标`
   - 搜索组合3：`公司名称 + OA系统 + 采购`
   - 搜索组合4：`公司名称 + 办公软件 + 中标`
   - 搜索组合5：`公司名称 + IT项目 + 招标公告`
   - 优先查看：最近12个月的结果

5. **行业招标信息平台**（免费，补充数据源）
   - 中国招标网：http://www.bidcenter.com.cn
   - 中国采购与招标网：http://www.chinabidding.cn
   - 搜索关键词：`公司名称`、`办公系统`、`协同软件`

6. **天眼查/企查查招投标**（付费，可选）
   - 仅用于重大项目验证和历史项目回溯
   - 注意成本控制

**关键信息提取**：
- 项目名称、采购单位、预算金额、投标截止日期
- 采购内容、采购方式、项目状态
- 联系信息、WPS 365匹配度评估

**6.2 子公司信息查询**

**触发条件**（满足任一即触发）：
1. 用户明确要求："帮我查一下XX公司的子公司"
2. 企业性质为集团（从基础信息判断）
3. 公司规模>1000亿：大型集团，必有子公司
4. `include_subsidiaries` 参数为 `true`

**数据获取方法**（按优先级）：
1. **公司官网**（免费，主要数据源）
   - 投资者关系板块：查找年报、企业集团构成
   - 公司治理板块：查找控股子公司、参股公司
   - 组织架构页面：查找组织结构图
   - WebReader配置示例：
   ```json
   {
     "url": "https://www.cmbchina.com/IR/",
     "return_format": "markdown",
     "retain_images": false,
     "timeout": 30,
     "with_links_summary": true
   }
   ```

2. **WebSearch**（免费，辅助数据源）
   - 搜索关键词：`公司名称 + 子公司`、`公司名称 + 控股子公司`
   - 搜索最新动态：`公司名称 + 最新新闻`
   - 优先看最近3个月结果

3. **巨潮网年报**（免费，备用数据源）
   - 年报"企业集团构成"章节
   - 提取子公司名称、持股比例、注册资本、业务范围

4. **天眼查/企查查API**（付费，可选）
   - 仅用于官网和年报信息不足时
   - 注意成本控制

**关键信息提取**：
- 子公司名称、持股比例、注册资本
- 业务范围、业务规模、IT投入规模
- WPS 365匹配度、跟进优先级评估
