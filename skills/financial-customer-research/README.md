# WPS 365 金融行业客户研究技能

为 WPS 365 售前团队提供专业的金融行业客户研究支持，自动生成结构化分析报告和解决方案匹配。

## 技能概述

本技能帮助售前人员在输入金融机构名称、部门或岗位时，自动生成包含**关键摘要洞察**、6张结构化分析表格和完整的研究报告，包括：

**关键摘要洞察**（销售/售前必读）：
- 机会评估：市场规模、紧迫性、机会评级
- 现有合作基础：是否有WPS 365合作
- 关键决策人观点：与WPS 365的契合度
- 客户痛点：明确的痛点列表和匹配度
- 竞争优势：对比主要竞争对手
- 关键风险：风险预警和应对策略

**6张结构化分析表格**：
1. **基础客户情况表**：公司规模、业务结构、战略方向、IT投入等
2. **BU业务情况表**：部门定位、业务规模、核心业务、KPI指标等
3. **部门业务逻辑表**：业务流程、协作关系、协作痛点、工具现状等
4. **决策链条表**：关键人物、决策角色、关注点、痛点、沟通策略等（包含详细履历和公开言论）
5. **招投标信息表**（可选）：最近的招标公告、中标结果、采购需求、预算范围、信息来源URL
6. **子公司情况表**（可选）：子公司名单、业务范围、规模、IT投入

## 使用方法

### 触发条件

当用户提供以下任一信息时，启动本技能：

- **公司名称**：如"招商银行"、"中国平安"、"中信证券"
- **公司简称**：如"招行"、"平安"
- **部门名称**：如"零售金融部"、"寿险事业部"
- **岗位/角色**：如"CIO"、"IT总监"
- **行业关键词**：如"银行业"、"保险业"

### 使用示例

#### 示例1：完整信息研究

```
用户：帮我研究"招商银行零售金融部"

助手：
1. 确认研究目标：招商银行 + 零售金融部
2. 数据获取：巨潮网年报 → 招商银行官网 → WebSearch 补充
3. 生成4张表格
4. 识别痛点并匹配 WPS 365 解决方案
5. 输出完整报告
```

#### 示例2：行业研究

```
用户：帮我研究"保险业的协同办公痛点"

助手：
1. 调用 financial-industry-pain-points.md
2. 提取保险业痛点
3. 匹配 WPS 365 解决方案
4. 提供同行业案例参考
```

#### 示例3：岗位研究

```
用户：帮我了解"银行CIO的关注点"

助手：
1. 调用 decision-chain-framework.md
2. 提取CIO角色的关注点和痛点
3. 提供沟通策略建议
4. 匹配 WPS 365 价值主张
```

## 数据获取流程

本技能按以下优先级获取客户信息：

1. **用户上传文件**（最优先）- 检查用户是否上传了客户资料、招标文件
2. **巨潮网公告**（权威来源）- 获取年报、半年度报告、重大事项公告
3. **公司官网**（官方渠道）- 了解业务介绍、组织架构、新闻动态
4. **行业协会**（专业网站）- 了解行业趋势、监管要求、痛点
5. **WebSearch**（补充验证）- 搜索最新网络信息
6. **实时信息查询**（可选触发）- 招投标信息、子公司信息

### 实时信息功能（新增）

#### 招投标信息查询

**触发条件**（满足任一即触发）：
- 用户明确要求："帮我查一下XX银行的招投标信息"
- `research_depth` 为 `comprehensive`：自动查询最近6个月招投标信息
- IT投入>1亿/年：大客户，值得深入挖掘
- `include_bidding` 参数为 `true`

**数据源**：
- 中国政府采购网（免费）：http://www.ccgp.gov.cn
- 各省市政府采购网（免费）：深圳、上海、北京等
- 天眼查招投标API（付费，可选）

**关键信息**：
- 项目名称、采购单位、预算金额、投标截止日期
- 采购内容、采购方式、项目状态、WPS 365匹配度

#### 子公司信息查询

**触发条件**（满足任一即触发）：
- 用户明确要求："帮我查一下XX公司的子公司"
- 企业性质为集团（从基础信息判断）
- 公司规模>1000亿：大型集团，必有子公司
- `include_subsidiaries` 参数为 `true`

**数据源**：
- 公司官网（免费，主要）：投资者关系板块、公司治理板块
- WebSearch（免费，辅助）：搜索"公司名称 + 子公司"
- 巨潮网年报（免费，备用）：年报"企业集团构成"章节
- 天眼查/企查查API（付费，可选）

**关键信息**：
- 子公司名称、持股比例、注册资本、业务范围
- IT投入规模、技术栈、决策链条、WPS 365匹配度

## 核心特性

### 1. 自动痛点匹配

基于关键词自动将客户痛点映射到 WPS 365 解决方案：

| 关键词 | 匹配方案 |
|--------|----------|
| 文档版本、版本管理 | WPS 365 协同文档 |
| 协同、协作 | 金山协作 IM + 多维表格 |
| 安全、合规 | 七重权限管理 + 私有化部署 |
| 国产化、信创 | 全栈信创支持 |
| AI、效率 | WPS AI 企业版 |

### 2. 决策链分析

自动分析决策链条中的关键人物：

- **决策者**（CIO、CEO）：关注ROI、战略价值、案例
- **评估者**（IT总监）：关注技术方案、集成、安全
- **影响者**（业务负责人）：关注业务价值、易用性
- **使用者**（一线员工）：关注操作体验、功能

### 4. 完整报告输出

生成包含以下内容的专业报告：

- 执行摘要（核心发现、合作机会评估）
- 4-6张结构化表格（带数据来源和日期，实时信息表可选）
- WPS 365 解决方案匹配（痛点-方案映射、推荐方案包、成功案例）
- 行动建议（近期/中期/长期）
- 数据来源说明（一手数据、二手数据、时效性说明）
