# "电梯演讲"生成器 (presale-elevator-pitch)

生成 30 秒销售话术，基于金字塔原理，提供焦虑/贪婪/愿景三套钩子，针对不同角色和行业定制。

## 快速开始

**触发条件**：
- 用户需要生成短话术
- 用户面对不同角色和行业的客户
- 用户需要快速打动对方

**使用示例**：
```
给我一个向 CEO 介绍的话术
金融行业 CIO 的 30 秒话术
焦虑版话术怎么写？
```

## 核心功能

1. **金字塔原理**：结论先行/论据支撑/行动号召
2. **三套钩子**：焦虑版/贪婪版/愿景版
3. **角色定制**：CEO/CIO/业务负责人
4. **口语化转换**：书面语 → 口语化

## 输出示例

```
焦虑版（80-100字）：
"张总，数字化转型只做了一半，数据安全出问题会被问责..."

贪婪版（80-100字）：
"张总，一套方案让全公司效率提升30%，一年省500万..."

愿景版（100-120字）：
"张总，打造成行业数字化标杆..."
```
