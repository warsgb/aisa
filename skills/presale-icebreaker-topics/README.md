# 破冰话题库 (presale-icebreaker-topics)

WPS 365 售前破冰话题库技能，用于陪领导应酬、和客户吃饭等社交场合。基于地域文化、行业趋势、角色特点，生成安全连接的破冰话题。

## 技能简介

本技能提供四大话题类型：

1. **地域文化话题**：结合当地的文化典故或美食
2. **行业正能量话题**：讨论行业的正面趋势、政策利好、技术创新
3. **角色导向话题**：根据角色关注点选择话题
4. **万能话题**：适用于任何场景的��用话题

## 快速开始

### 基础用法

```
"和上海金融客户吃饭，准备些破冰话题"
"北京政府客户商务晚餐的话题"
"茶歇时和CIO聊什么"
```

### 完整参数

```
使用 presale-icebreaker-topics，客户地区：上海，客户行业：金融，
客户角色：CIO，应酬场景：商务午餐
```

## 参数说明

| 参数 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| customer_location | string | 否 | 客户所在城市/地区 | "上海"、"北京" |
| customer_industry | string | 否 | 客户行业 | "金融"、"政府"、"制造业" |
| customer_role | string | 否 | 客户角色 | "CEO"、"CIO"、"业务负责人" |

| avoid_topics | array | 否 | 需要回避的话题 | ["股市", "房价"] |

## 使用建议

### 核心原则

1. **抛砖引玉**：话题是引子，目的是让客户多说
2. **倾听为主**：80%倾听，20%表达
3. **灵活切换**：一个话题冷场，立即切换
4. **真诚赞美**：适时赞美客户或客户所在城市/公司
5. **避免说教**：不要搞个人演讲，保持对话平衡

### 敏感话题红线

| 禁止类别 | 具体话题 |
|---------|---------|
| 政治敏感 | 时事政治、领导人评价、国际关系 |
| 宗教信仰 | 宗教观点、民族话题 |
| 个人隐私 | 收入、家庭矛盾、健康状况 |
| 负面新闻 | 客户公司负面、行业丑闻 |
| 竞品对比 | 直接批评竞品 |
| 价格讨论 | 具体报价、折扣 |
| 股市房价 | 股票涨跌、房价走势 |

## 常见问题

### Q: 破冰话题失败了怎么办？

A: 保持轻松，立即切换话题。可以转向万能话题如天气、旅游、美食等。

### Q: 客户主动提起敏感话题怎么办？

A: 礼貌倾听，不发表明确意见，适时转移话题。如果客户坚持，可以表示理解但不深入。

### Q: 什么时候转入正式话题？

A: 观察客户反应，当氛围轻松、客户表达意愿较强时，可以自然过渡到业务话题。

### Q: 如何判断话题是否合适？

A: 检查三点：1) 是否安全无敏感内容 2) 是否低认知门槛 3) 是否能让客户多说
