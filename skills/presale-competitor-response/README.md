# 竞品"杀手锏"反制 (presale-competitor-response)

应对竞品"杀手锏"，采用 FUD 策略（捧杀-转折-重构），针对不同客户类型提供差异化反制策略。

## 快速开始

**触发条件**：
- 客户提到竞品优势
- 客户对比竞品
- 需要应对竞品话术

**使用示例**：
```
客户说微软 365 更好用，怎么回应？
飞书和 WPS 365 的区别是什么？
```

## 核心功能

1. **竞品优势分析**：客观分析竞品优势
2. **FUD 策略应用**：捧杀-转折-重构
3. **差异化反制**：国企/金融/政府/民企不同策略
4. **生成反制话术**：初次接触/深入沟通/应对质疑

## 输出示例

```
竞品对比表：
| 对比维度 | WPS 365 | 微软 365 | 飞书 | 钉钉 |

FUD 策略应用：
捧杀：Office 产品确实成熟...
转折：不过在企业场景...
重构：我们的全栈信创支持...
```
