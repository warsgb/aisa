# Test Skill

A test skill for verification of skill loading system.

## 功能说明

这是一个测试技能，用于验证技能加载系统是否正常工作。

## 使用场景

当系统需要验证以下功能时使用：
1. 技能是否从文件系统正确加载
2. Frontmatter 元数据是否正确解析
3. 技能是否正确保存到数据库
4. 技能执行流程是否正常工作

## 使用方法

在技能中心选择"Test Skill"，输入任意测试文本，执行后应收到确认响应。

## 预期输出

```
✓ 技能加载系统工作正常
✓ 输入已接收：{用户输入的文本}
```
