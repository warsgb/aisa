## 对话模拟

### 第一轮攻击

**客户（李总）**：今天时间不多，10分钟。你们这个私有化部署，核心是什么？

**售前**：李总，WPS 365 私有化部署的核心是数据完全自主可控。所有数据部署在您自己的服务器上，我们不接触任何数据。

**客户（李总）**：数据在我服务器上，但你们的软件呢？有没有后门？怎么保证没有安全问题？

**售前**：我们自研核心代码，30年技术积累，没有后门。而且我们通过了等保三级认证和涉密认证，安全有保障。

**客户（李总）**：等保三级、涉密认证？证书拿来看看。

**售前**：没问题，我可以提供完整的证书复印件。我们是行业内少有同时拥有等保三级和涉密认证的厂商。

**客户（李总）**：那信创呢？国资委79号文要求2027年全信创化，你们能满足吗？

**售前**：完全可以。我们适配信创三大架构：C86、ARM、自主指令集，符合信创工委会明确的技术路线要求。很多央企客户都在用我们的方案做信创改造。

**客户（李总）**：哪些央企？举个例子。

**售前**：湖北文旅是全省省属企业首个信创服务器私有化部署的集团级协同平台。还有海螺集团、中国信保、紫金投资等。

**客户（李总）**：那如果审计来查，你们能提供什么证明材料？

**售前**：我们提供完整的审计留痕功能，所有操作都有记录，包括文档查看、编辑、下载、分享等，可追溯可审计。

**客户（李总）**：这些审计记录存在哪？会不会被篡改？

**售前**：审计记录存储在私有化环境中，由您自己管理。我们采用不可篡改的存储方式，确保审计记录的真实性。

**客户（李总）**：万一数据泄露了，你们怎么负责？

**售前**：我们有七重权限管理，精细化控制谁能看、谁能编辑、谁能下载。还有端云一体不落地管控、5000个敏感词阻断能力。如果发生数据泄露，我们会配合调查并提供技术支持。

**客户（李总）**：配合调查？那不是你们的责任？

**售前**：如果是我们的产品漏洞导致的，我们会承担相应的责任。但我们这么多客户，还没有发生过数据泄露事件。

**客户（李总）**：那员工离职后，能不能保证他不带走数据？

**售前**：可以。离职员工账号自动回收，所有权限立即失效。移动端支持设备管控，可以远程擦除企业数据。

**客户（李总）**：远程擦除？这不会影响员工个人数据吗？有没有法律风险？

**售前**：我们的企业数据和个人数据是分离的，远程擦除只擦除企业数据，不影响个人数据。而且这需要员工入职时签署的设备管理协议授权，符合法律规定。

**客户（李总）**：嗯...那实施周期多长？我们时间紧，年底前要上线。

**售前**：如果时间紧，我们可以分阶段实施。先在核心部门试点，3个月上线，然后逐步推广。这样既能满足时间要求，又能控制风险。

**客户（李总）**：试点失败了怎么办？

**售前**：我们提供完整的试点方案，包括需求调研、方案设计、实施部署、培训上线。试点期间我们有专人驻场支持，确保成功。如果试点不成功，不需要继续推广，损失可控。

**客户（李总）**：那员工培训怎么办？这么多员工，培训周期会不会很长？

**售前**：我们的界面和Office类似，学习成本低。我们提供分层次培训：管理员培训、关键用户培训、全员培训。还有在线学习平台和视频教程，员工可以自主学习。

**客户（李总）**：学习成本低有多低？大概多久能用？

**售前**：大部分员工1-2天就能熟悉基本操作，1周内可以完全上手。我们有客户上线后，当月活跃度就达到90%以上。

**客户（李总）**：90%以上？哪个客户？

**售前**：中芯国际，文档中心月活95%+。中国信保，云文档月活占比90%。

**客户（李总）**：行吧。最后一个问题，价格怎么样？我们预算有限。

**售前**：我们支持信创改造，可以避免重复投入。对比M365，3年TCO降低超70%。而且我们有成本优化策略，比如蓝领账号拆分、利旧现有资源等，可以进一步降低成本。

**客户（李总）**：70%？怎么算的？

**售前**：这是基于实际项目对比的数据。M365三年订阅费用很高，而且不支持信创，需要额外投入。我们一次性部署，长期使用，综合成本更低。我可以提供详细的TCO对比报告。

**客户（李总）**：好，把TCO对比报告和案例材料发我，我看完让信息部门评估一下。

**售前**：没问题，李总。我今天就准备好材料发给您。另外，如果您需要，我可以安排技术专家和信息部门进行详细交流。

**客户（李总）**：先看材料吧，有需要再联系。

## 教练复盘分析

### 1. 回答逻辑分析

**优点**：
- 开门见山，直接回应了"核心是什么"的问题
- 对于每个合规相关问题都有明确的认证和证明
- 案例引用具体，有客户名称和数据支撑
- 对于风险问题提供了具体的技术措施

**问题**：
- 在"责任承担"问题上，表述有点保守，可能让客户觉得不够有信心
- "70% TCO降低"需要更详细的计算依据
- 对于"10分钟时间限制"的回应可以更高效

**改进建议**：
- 责任承担问题可以更自信，强调产品成熟度和零安全事故记录
- TCO计算应该准备详细的Excel表格，让客户可以自己调整参数
- 开场可以更简洁，快速切入核心合规问题

---

### 2. 情绪控制评估

**表现评估**：
- ✅ 专业冷静：始终保持专业态度
- ✅ 自信有力：回答时语气坚定
- ✅ 节奏控制：适应客户快节奏，没有拖沓

**改进建议**：
- 继续保持高效简洁的沟通风格
- 在客户说"时间不多"时，可以主动提出"我先说核心三点，您看哪个需要详细展开"

### 3. 话术参考

**更好的开场**（适应时间限制）：
```
"李总，明白您时间宝贵。我用3分钟说核心三点：
第一，数据完全自主可控，在您服务器上；
第二，有等保三级和涉密认证，审计留痕完整；
第三，信创全栈支持，符合79号文要求。
您看哪点需要我详细展开？"
```

**更好的回答**（关于责任承担）：
```
"我们对产品安全有充分信心。我们服务了100+家央企/国企，
从来没有发生过数据泄露事件。如果是我们的产品漏洞导致的问题，
我们会承担全部责任。但我们更强调的是，
我们的七重权限管理和端云一体不落地管控，
可以从根本上防范数据泄露风险。"
```

**更好的回答**（关于TCO计算）：
```
"70%的降低是基于三年总体拥有成本计算的。
M365三年订阅费用大约是XX元/人，而且不支持信创，
如果需要信创改造还需要额外投入。
我们是一次性部署费用+年度服务费，三年下来综合成本更低。
我有一份详细的Excel计算表，您可以根据咱们实际情况调整参数，
我可以发给您。"
```

**可引用的数据/案例**：
- 安全认证：等保三级、涉密认证、ISO 27001、ISO 20000
- 信创支持：适配三大架构（C86、ARM、自主指令集）
- 央企案例：100+家央企/国企，50+家金融机构
- 活跃度：中芯国际95%+，中国信保90%
- TCO对比：对比M365，3年TCO降低超70%

## 关键经验总结

1. **合规狂人客户特点**：
   - 时间紧张，需要高效沟通
   - 关注审计和合规，需要具体证明
   - 风险厌恶，需要提供风险控制措施
   - 要求有担责态度，不能模糊

2. **应对策略**：
   - 开场快速切入核心，适应客户时间限制
   - 准备所有合规认证的复印件
   - 强调审计留痕和风险控制能力
   - 用央企案例建立信任

3. **避坑指南**：
   - 不要在责任问题上模糊处理
   - 不要夸大效果，准备具体计算依据
   - 不要忽略客户时间限制，简洁高效
   - 费用问题要透明，提供详细计算表格

4. **成功关键**：
   - 认证齐全（等保三级、涉密认证）
   - 案例充分（央企案例、数据支撑）
   - 风险可控（权限管理、审计留痕）
   - 责任明确（敢于承担责任）
