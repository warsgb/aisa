# 表格4：决策链条表模板（教育行业）

## 用途

本模板用于记录教育机构的决策链条信息，为后续分析提供决策数据支撑。

## 字段定义

| 字段 | 填写说明 | 数据来源 | 示例 |
|------|----------|----------|------|
| 姓名 | 关键人物姓名 | 访谈/官网 | 张三 |
| 职位 | 具体职位 | 学校官网/LinkedIn | 党委书记 / 校长 |
| 决策角色 | 决策者/评估者/影响者/使用者 | 访谈/分析 | 决策者 |
| 决策权限 | 预算/技术/业务决策权 | 访谈 | 预算决策权500万以内 |
| 关注点 | 最关注的方面 | 访谈/行业 | 双一流建设、智慧校园、国产化 |
| 关键痛点 | 核心痛点和挑战 | 访谈/观察 | 系统分散、协作效率低 |
| 痛点严重度 | 1-5分（5最严重） | 访谈/分析 | 4分 |
| WPS 365价值 | WPS 365如何解决 | 分析 | 统一平台提升效率30% |
| 沟通策略 | 推荐的沟通策略 | 分析 | 强调双一流、智慧校园、国产化 |
| 沟通渠道 | 建议的沟通渠道 | 访谈/行业 | 先通过信息中心主任接触 |
| 数据来源 | 数据具体来源 | - | 内部访谈 |
| 更新日期 | 数据获取日期 | - | 2026-02-27 |

## 教育行业决策链分析

### 高等教育决策链
```
党委（最终决策）→ 校长/校长办公会 → 分管副校长 → 职能部门（教务处/科研处/信息化中心）→ 院系 → 教师/学生
```

### K12基础教育决策链
```
教育局/教委 → 校长 → 分管副校长 → 教务处/总务处/信息中心 → 年级组/教研组 → 教师/家长
```

### 职业教育决策链
```
校长/校长办公会 → 分管副校长 → 职能部门（教务处/实训中心/信息中心）→ 专业系部 → 教师/学生
```
