# 关键摘要洞察模板（教育行业）

**说明**：本模板用于在报告开头为销售/售前团队提供关键洞察，帮助快速了解教育行业客户价值和机会。

**使用场景**：
- 位于报告开头，在"执行摘要"之前
- 为销售/售前团队提供5分钟快速决策信息
- 不包含行动计划、销售策略等实施细节
- 只包含核心洞察和事实

## 使用指南

### 必填项（核心8点）
1. ✅ **机会评估**：市场规模、紧迫性、采购周期、机会评级
2. ✅ **现有合作基础**（或市场切入点）
3. ✅ **关键决策人观点**：至少3个核心观点，标注与WPS 365匹配度
4. ✅ **客户痛点**：至少3个痛点，标注WPS 365匹配度
5. ✅ **竞争优势**：对比主要竞争对手
6. ✅ **次要决策人/细分场景**（可选，视情况）
7. ✅ **分校区/院系机会**（教育行业特有，视情况）
8. ✅ **关键风险**：至少3个风险及应对，包括教育行业特殊风险

### 评分标准（机会评级）
| 评分 | 说明 |
|------|------|
| ⭐⭐⭐⭐⭐ 五星级 | 市场规模大（学生数10000+）、紧迫性高（寒暑假采购节点）、决策人观点契合、有合作基础 |
| ⭐⭐⭐⭐ 四星级 | 市场规模大（学生数5000-10000）、或紧迫性高、或决策人观点契合 |
| ⭐⭐⭐ 三星级 | 市场规模中等（学生数1000-5000），需要评估机会 |
| ⭐⭐ 二星级 | 市场规模小（学生数<1000），或低优先级 |

### 教育行业特殊关注点

**双一流建设**：
- 双一流高校关注双一流学科建设
- 教学资源库建设是重要切入点

**智慧校园建设**：
- K12学校关注智慧校园建设
- 智能备课、学情分析是重点

**信创国产化**：
- 公办院校有信创国产化要求
- 全栈信创适配是核心优势

**寒暑假采购**：
- 教育行业采购集中在寒暑假（1-2月、7-8月）
- 错过采购窗口需等下个学期

### 质量检查清单

完成关键摘要洞察后，进行以下检查：

- [ ] **简洁明了**：每点1-3句话，不超过半页
- [ ] **数据支撑**：每个观点都有数据或原文引用支撑
- [ ] **战略高度**：聚焦战略价值，不陷入细节
- [ ] **销售导向**：帮助销售快速判断是否跟进
- [ ] **可信度高**：来源明确，不夸大
- [ ] **可执行性**：洞察能够指导下一步行动
- [ ] **教育行业特性**：包含双一流、智慧校园、信创、采购周期等教育行业特性
