# 表格6：分校区情况表模板（教育行业）

## 用途

本模板用于记录教育机构的分校区情况，为后续分析提供分校区数据支撑。

## 字段定义

| 字段 | 填写说明 | 数据来源 | 示例 |
|------|----------|----------|------|
| 分校区名称 | 分校区具体名称 | 学校官网/年报 | 东校区 |
| 分校区定位 | 在学校中的定位 | 学校官网/年报 | 主校区/教学区 |
| 所在地 | 分校区注册地或所在地 | 学校官网/官网 | 北京市朝阳区 |
| 业务范围 | 分校区主营业务和经营范围 | 学校官网/年报 | 本科教学、继续教育 |
| 业务规模 | 分校区学生/师资规模 | 年报/官网 | 学生2000+、师资100+ |
| IT投入规模 | 分校区年度IT投入（估算） | 年报/访谈 | 年度IT投入约100万 |
| IT团队规模 | 分校区IT团队人数 | 招聘/访谈 | IT团队20人 |
| 技术栈 | 分校区主要技术架构 | 招聘/技术社区 | Java、微服务、云原生 |
| 决策链条 | 分校区IT决策路径 | 访谈/分析 | 信息中心主任→技术负责人 |
| 现有IT系统 | 分校区主要IT系统 | 访谈/招标 | 教务系统、OA系统 |
| WPS 365匹配度 | 匹配程度（高/中/低） | 分析 | 高 |
| 跟进优先级 | 跟进优先级（高/中/低） | 分析 | 高（主校区） |
| 数据来源 | 数据具体来源，包括URL | - | 学校官网 |
| 更新日期 | 数据获取和更新日期 | - | 2026-02-27 |
