# 教育行业客户深度研究技能 (education-customer-research)

## 技能概述

本技能为 WPS 365 售前团队提供专业的教育行业客户研究支持，能够自动识别客户类型（高等教育/K12/职业教育），生成结构化分析报告，并匹配 WPS 365 解决方案。

## 支持的客户类型

### 高等教育
- **双一流高校**：如清华大学、北京大学、深圳大学
- **省属本科**：如四川农业大学、吉林化工大学
- **继续教育学院**：如清华大学继续教育学院、武汉大学非学历教育

### 基础教育（K12）
- **教育局/教委**：如海淀区教委、乐清市教育局
- **中小学**：如北京四中、上海中学
- **一贯制学校**：小学到高中全覆盖

### 职业教育
- **高职专科**：如九江职业大学、深圳职业技术学院
- **中职技校**：各类中等职业学校、技校

### 教培机构
- **培训机构**：各类教育培训机构
- **国际学校**：民办国际学校

## 核心功能

### 1. 客户类型自动识别
- 根据学校名称自动识别办学层次
- 根据部门名称自动识别业务类型
- 根据关键词自动识别教育场景

### 2. 六张核心表格
- **基础学校情况表**：学校规模、层次、主管部门、IT投入等
- **学术业务情况表**：院系业务、课程设置、教学资源等
- **部门业务逻辑表**：业务流程、协作关系、现有工具等
- **决策链条表**：决策角色、关注点、沟通策略等
- **招投标信息表**（可选）：采购项目、预算、时间节点等
- **分校区情况表**（可选）：各校区规模、IT投入、优先级等

### 3. 痛点智能匹配
- 基于关键词自动匹配教育行业痛点
- 引用详细的教育行业痛点库
- 匹配对应的 WPS 365 解决方案

### 4. 案例智能推荐
- 基于11个真实教育行业案例
- 自动推荐相似客户案例
- 提供可量化的价值证明

### 5. 完整研究报告
- 关键摘要洞察（8个核心洞察点）
- 执行摘要
- 6张核心表格
- WPS 365 解决方案匹配
- 行动建议

## 使用方式

### 基础用法
```
帮我研究"清华大学继续教育学院"
```

### 高级用法
```
帮我研究"海淀区教委信息化中心"，了解智慧校园建设情况
```

### 完整用法
```
帮我研究"九江职业大学实训中心"，目标是了解产教融合需求，面向信息中心主任
```

## 教育行业特有功能

### 1. 采购周期识别
- 自动识别寒暑假采购节点（1-2月、7-8月）
- 预算周期识别（12月制定年度预算）

### 2. 双一流建设关注
- 自动识别双一流高校
- 推荐教学资源库、科研协同方案

### 3. 信创国产化关注
- 自动识别公办院校信创要求
- 推荐全栈信创适配方案

### 4. 多校区/院系机会
- 自动识别分校区情况
- 推荐分阶段实施策略
