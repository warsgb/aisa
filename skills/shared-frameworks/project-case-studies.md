# 项目案例库

本文件为 WPS 365 售前工具提供按行业/场景分类的详细案例库，供各个技能快速检索和引用。

---

## 使用说明

### 案例结构

每个案例包含以下字段：
- **客户**：客户名称
- **行业**：行业-子行业
- **规模**：用户数量/分支机构数量/业务规模
- **建设内容**：主要建设内容概述
- **应用效果**：量化效果数据
- **客��评价**：客户反馈或价值总结

### 引用方式

各技能可通过以下方式检索案例：
- 按行业筛选：金融、政府与国企、能源、制造、军工等
- 按痛点筛选：系统割裂、数据安全、信创合规等
- 按价值筛选：降本增效、效率提升、合规保障等
