## 筹码类型库

### 服务类筹码

**核心逻辑**：削减高成本服务，改用低成本替代方案。边际成本低，客户感知价值适中。

#### 详细筹码表

| 筹码名称 | 我方成本 | 客户价值 | 适用场景 | 风险提示 |
|---------|---------|---------|---------|---------|
| **削减现场人天** | 节省差旅成本（机票、酒店、餐饮） | 仍获得专业支持，响应及时 | 非关键阶段、客户有一定运维能力 | 需评估远程支持是否满足需求 |
| **减少培训场次** | 可复用内容，讲师时间节省 | 灵活学习时间，可反复观看 | 培训预算紧张、学员时间分散 | 需提供优质视频教程和学习平台 |
| **缩短驻场周期** | 降低人力成本（驻场补贴） | 获得关键节点现场保障 | 客户有基础运维能力 | 关键节点（上线、验收）需现场 |
| **减少定制开发** | 避免项目风险、开发成本 | 系统更稳定、升级更容易 | 需求可标准化实现 | 需确认标准功能满足需求 |
| **增加自助服务** | 减少人工成本 | 客户自主掌控、响应更快 | 客户有IT团队 | 需提供完善的自助服务文档 |
| **延迟服务响应** | 优化人力调配 | 价格降低，SLA可接受 | 非关键业务场景 | 需明确SLA，避免影响关键业务 |

#### 服务类筹码组合策略

**降本套餐**：
- 削减30%现场人天 + 减少培训场次 → 可降8-10%
- 缩短50%驻场周期 + 增加自助服务 → 可降10-12%

**适用话术**：
```
"张总，咱们在服务交付方式上调整一下如何？
原计划20人天现场支持，我们可以削减为10人天，其余改为远程专家支持。
这样既为您节省了差旅成本，我们也能在价格上给您更大优惠。"
```

### 品牌类筹码

**核心逻辑**：用品牌曝光、标杆案例价值交换价格。我方成本主要是营销资源，客户获得品牌价值。

#### 详细筹码表

| 筹码名称 | 我方成本 | 客户价值 | 适用场景 | 风险提示 |
|---------|---------|---------|---------|---------|
| **标杆案例授权** | 品牌背书（营销资源） | 行业标杆地位、邀功资本 | CEO/总经理邀功需求强烈 | 需法务审核授权条款 |
| **联合新闻稿** | PR资源、媒体渠道 | 媒体曝光度、行业影响力 | 需要宣传项目成果 | 新闻稿内容需双方审核 |
| **高层互访** | 高层时间（半天） | 提升合作层级、建立高层关系 | 战略合作项目、大客户 | 需协调高层时间 |
| **行业峰会邀请** | 门票+VIP席位+演讲机会 | 行业影响力、社交价值 | 业务负责人、技术负责人 | 需确认客户有参会意愿 |
| **案例拍摄合作** | 拍摄团队、剪辑成本 | 高质量宣传视频、品牌展示 | 客户需要品牌宣传素材 | 需确认拍摄内容和展示范围 |
| **联合白皮书** | 研究撰写成本 | 行业思想领导力、专业形象 | 行业领先企业 | 需确保内容符合双方定位 |

#### 品牌类筹码组合策略

**品牌套餐**：
- 标杆案例授权 + 联合新闻稿 → 可降8-12%
- 标杆案例 + 行业峰会VIP → 可降10-15%
- 全套品牌方案（案例+新闻稿+峰会+白皮书）→ 可降15-20%

**适用话术**：
```
"张总，如果您能签署标杆案例授权，允许我们将贵公司作为行业标杆案例进行宣传，
我可以在价格上给您更大优惠。而且咱们可以联合发布新闻稿，
您也能在行业峰会上作为VIP嘉宾分享经验。这对您向上级汇报邀功非常有帮助。"
```

### 账期类筹码

**核心逻辑**：用优惠现金流条件，换取价格让渡。我方获得现金流改善或长期锁定，客户获得灵活付款。

#### 详细筹码表

| 筹码名称 | 我方收益 | 客户价值 | 适用场景 | 风险提示 |
|---------|---------|---------|---------|---------|
| **预付全款折扣** | 现金流改善、减少坏账风险 | X%折扣（通常5-10%） | 客户现金充裕、年度预算充足 | 需确认客户有支付能力 |
| **长期合约优惠** | 锁定长期收入、降低获客成本 | 签3年送6个月、锁定价格 | 长期合作意向、担心未来涨价 | 需评估长期合约价格风险 |
| **分期付款方案** | 可能收利息、扩大客户范围 | 降低单期压力、可立项 | 预算分期、现金流紧张 | 需评估客户信用风险 |
| **季度付款优惠** | 现金流稳定、匹配客户节奏 | 按季度支付、财务流程顺畅 | 财务规范、季度预算制 | 需明确逾期处理条款 |
| **渐进式付款** | 早期现金流改善 | 首付少、尾款多 | 客户前期预算紧张 | 需确保尾款回收有保障 |
| **里程碑付款** | 降低项目风险 | 按进度付款、风险共担 | 大型项目、实施周期长 | 需明确里程碑标准和验收 |

#### 账期类筹码组合策略

**现金流套餐**：
- 预付全款 → 可降5-8%
- 签3年送6个月 → 年化价格降约14%
- 分期付款（4期）+ 利息 → 价格不变，但客户可立项

**适用话术**：
```
"张总，如果咱们能把合作周期拉长，我就能给您更大的优惠。
签3年送6个月，相当于您用3年的预算享受3.5年的服务。
而且如果年度一次性付款，还能再享受额外折扣，总共能降XX%。"
```

### 产品组合筹码

**核心逻辑**：结合历史已购产品，设计打包优惠方案。提升客单价，客户获得综合优惠。

#### 详细筹码表

| 筹码名称 | 我方成本 | 客户价值 | 适用场景 | 风险提示 |
|---------|---------|---------|---------|---------|
| **老客户续费优惠** | 锁定续费、降低流失率 | 已购产品续费折扣 | 续费谈判 | 需评估续费折扣对毛利影响 |
| **增购模块打包** | 提升客单价、交叉销售 | 新老模块打包价 | 功能扩容、新需求 | 需确认模块兼容性 |
| **用户数扩容优惠** | 规模效应、降低边际成本 | 阶梯折扣、批量优惠 | 用户数增长、组织扩容 | 需设定合理的阶梯折扣 |
| **跨产品打包** | 生态协同、提升粘性 | WPS 365 + 其他产品组合 | 多产品需求、一站式采购 | 需协调产品线和定价 |
| **版本升级优惠** | 销售高版本、提升ARPU | 升级折扣、获得新功能 | 版本升级谈判 | 需确保升级价值明显 |
| **行业解决方案包** | 深度行业定制、差异化 | 行业模板、最佳实践 | 行业特征明显 | 需投入行业方案研发 |

#### 产品组合筹码具体策略

**老客户续费优惠**：
- 已购WPS专业版 → 续费享8折，同时购买WPS 365享组合价
- 已购金山云 → 联合采购享9折

**增购模块打包**：
- 基础版 + 增值模块（审批、表单、BI）→ 打包优惠15%
- 按用户数阶梯：100用户以下9折，100-500用户85折，500+用户8折

**跨产品打包**：
- WPS 365 + 金山云存储 → 组合优惠10%
- WPS 365 + 金山办公安全 → 组合优惠12%

**适用话术**：
```
"张总，我看贵公司已经采购了WPS专业版，咱们可以设计一个组合方案：
专业版续费享受8折，同时新购WPS 365还能享受组合优惠。
这样您既解决了续费问题，又获得了升级功能，总体成本更优。"
```

## 谈判话术模板库

### 标准话术框架

#### 1. 四步对话法

**第一步：认可**
- 目的：建立共情，降低防御
- 话术：
  - "我完全理解您的预算压力..."
  - "您的要求我听到，咱们一起想想办法..."
  - "价格确实是重要因素，咱们看看有没有其他方案..."

**第二步：转折**
- 目的：引导思维转换，扩大谈判蛋糕
- 话术：
  - "如果您能在XX方面支持我们，我们可以在价格上..."
  - "咱们换个思路，您看这样行不行..."
  - "我有个提议，您看看能不能接受..."

**第三步：价值量化**
- 目的：说明客户获得的价值，增强吸引力
- 话术：
  - "这样您既解决了预算问题，又获得了XX价值..."
  - "这个方案对您来说，总价值是..."
  - "从TCO角度看，您实际上节省了..."

**第四步：行动呼吁**
- 目的：推动决策，明确下一步
- 话术：
  - "您看这个方案怎么样？"
  - "咱们先从这个方案开始，细节再细化..."
  - "如果您同意这个方向，我马上准备方案..."

### 分类话术模板

#### 服务类筹码话术

**削减现场人天**：
```
"张总，我理解您的预算压力。咱们在服务交付方式上调整一下如何？
原计划20人天现场支持，我们可以削减为10人天，其余改为远程专家支持。
这样既为您节省了差旅成本，我们也能在价格上给您8%的折扣。

远程支持其实响应更及时，而且我们的专家团队随时待命。
从成本角度看，您既节省了差旅费，又获得了价格优惠，总体更划算。

您觉得这个方案怎么样？"
```

**减少培训场次**：
```
"李总，关于培训这块，咱们可以优化一下。
原计划5场现场培训，我们可以改为2场现场+3场线上录播课程。
这样既节省了您的组织成本，学员还能灵活安排时间反复学习。

作为交换，我们可以在价格上给您5%的折扣。您看这样行吗？"
```

```
您看这个方案怎么样？如果同意，我马上准备授权书和新闻稿草案。"
```

**高层互访**：

### 紧急情况话术

**客户强硬要求降价**：
```
"张总，我理解您的要求。但咱们现在的价格已经是公司授权的最低价了，
再降就突破毛利红线了。

不过，咱们可以换个思路。如果您能在XX方面（如签署标杆案例授权）支持我们，
我们可以在其他方面给您补偿。这样您既获得了XX价值，价格也在可接受范围内。

您看这个方案怎么样？如果还有其他想法，咱们再商量。"
```

**客户威胁选择竞品**：

## 心理学原理库

### 核心心理学原理详解

#### 1. 互惠原理 (Reciprocity)

**机制**：
- 给予对方某物，对方会感到有回报的义务
- 人们不喜欢亏欠他人，会主动寻求平衡

**应用场景**：
- 服务换折扣：削减服务为客户节省成本，客户感到应回报
- 增值服务赠送：赠送培训、咨询服务，客户更易接受价格

**话术应用**：
```
"我们愿意削减30%的现场人天，改为远程支持，这为您节省了XX万元成本。
作为交换，也希望您在价格上能支持我们的底线。"
```

**注意事项**：
- 我方付出必须是"真实的"，客户能感知到价值
- 不要先提要求，先给予再要求回报

#### 2. 社会认同 (Social Proof)

**机制**：
- 人们会跟随类似人群的行为
- "别人都这么做"会产生从众心理

**应用场景**：
- 竞品对比："XX行业标杆企业也是这样合作的"
- 案例引用："某银行客户就是这样部署的"

**话术应用**：
```
"XX行业的头部企业，像XX公司、XX公司，都是采用标杆案例合作的方式。
他们既获得了行业标杆地位，又享受了价格优惠。"
```

**注意事项**：
- 引用的案例必须真实可信
- 案例与客户要有相似性（行业、规模）

#### 3. 损失厌恶 (Loss Aversion)

**机制**：
- 人们对损失的敏感度是对同等收益的2倍
- "限时优惠"利用人们害怕错过的心理

**应用场景**：
- 限时折扣："这个优惠只有本周有效"
- 限量名额："这个方案只有3个名额"

**话术应用**：
```
"这个长期合约优惠只有本周有效，如果您今天签约，
可以享受签3年送6个月的优惠。如果错过，就损失了6个月的价值。"
```

**注意事项**：
- 限时必须真实，不能虚假承诺
- 适可而止，过度使用会引发反感

#### 4. 锚定效应 (Anchoring)

**机制**：
- 第一印象影响后续判断
- 人们会以第一个信息为"锚点"进行对比

**应用场景**：
- 产品组合：用原价作为锚点，组合价显得便宜
- 折扣计算："原价XX万，现价XX万，您节省了XX万"

**话术应用**：
```
"如果我们单独计算，培训服务价值XX万，实施服务价值XX万，
现在打包在一起，总价XX万，您相当于节省了XX万。"
```

**注意事项**：
- 锚点价格必须合理，不能虚高
- 对比必须清晰，让客户看到实实在在的节省

#### 5. 稀缺性原理 (Scarcity)

**机制**：
- 稀缺的东西更有价值
- "物以稀为贵"的心理

**应用场景**：
- 限量名额："这个优惠只有3个名额"
- 限时方案："本季度只有这个方案"

**话术应用**：
```
"这个标杆案例合作方案，我们本季度只有3个名额。
目前已经确认了2个，还剩最后一个。您如果感兴趣，需要尽快确认。"
```

**注意事项**：
- 稀缺必须真实
- 不要过度使用，降低可信度

#### 6. 承诺一致性 (Commitment and Consistency)

**机制**：
- 人们希望保持言行一致
- 一旦做出小承诺，更容易接受大要求

**应用场景**：
- 先让客户认同小观点，再引导到价格
- "您也认同效率提升很重要对吧？"

**话术应用**：
```
"张总，您也认同效率提升对贵公司很重要，对吧？
WPS 365平均能提升30%效率，一年节省XX万成本。
既然效率提升这么重要，咱们在价格上能不能各退一步？"
```

**注意事项**：
- 小承诺必须真实有效
- 不要强迫客户表态

### 心理学原理组合应用

**组合策略1：锚定 + 损失厌恶**
```
"原价100万，现在优惠价85万（锚定）。
这个优惠只有本周有效，错过就损失15万（损失厌恶）。"
```

**组合策略2：社会认同 + 稀缺性**
```
"XX行业标杆企业都是这样合作的（社会认同）。
本季度只有3个名额，还剩1个（稀缺性）。"
```

**组合策略3：互惠 + 承诺一致性**
```
"我们先赠送您3个月免费试用（互惠）。
您试用后如果认可，咱们再谈正式合作（承诺一致性）。"
```

## 红线警示

### 财务红线

**核心原则**：
- ⚠️ 任何筹码交换都不能突破公司毛利红线
- ⚠️ AI 不能代替算账，价格计算需经财务确认
- ⚠️ 折扣权限不能超过个人授权范围

**财务确认清单**：
- [ ] 毛利红线：方案不突破公司最低毛利率要求
- [ ] 现金流影响：账期方案需财务评估现金流影响
- [ ] 成本核算：服务削减需评估实际成本节省
- [ ] 折扣权限：确认折扣在个人授权范围内

**价格计算警示**：
```
AI 可以提供筹码方案和话术，但具体价格计算必须由财务确认。
AI 说"可降10%"只是建议，实际数字需要财务核算后确定。
```

### 法律风险

**品牌类筹码注意事项**：
- [ ] 授权条款：案例授权书需法务审核
- [ ] Logo使用：确认客户Logo使用授权范围
- [ ] 新闻稿内容：联合新闻稿需双方审核
- [ ] 隐私保护：注意客户敏感信息保护

**合同条款警示**：
- [ ] 付款条款：分期付款需明确逾期处理
- [ ] 服务条款：服务削减需明确SLA变化
- [ ] 违约责任：长期合约需明确违约责任
- [ ] 知识产权：定制开发需明确知识产权归属

### 实施风险

**服务类筹码风险**：
- [ ] 实施风险：削减现场支持需评估实施成功率
- [ ] 培训效果：减少培训场次需提供替代学习方案
- [ ] 定制开发：减少定制需确认需求可用标准功能满足
- [ ] 运维责任：缩短驻场需明确运维责任边界

**风险话术**：
```
"张总，关于削减现场人天这个方案，我需要和实施团队确认一下，
确保远程支持能够满足贵公司的需求。咱们先从技术方案评估开始。"
```

### 职业道德

**诚信原则**：
- [ ] 不虚假承诺：不能承诺无法兑现的条件
- [ ] 不误导客户：不能隐瞒重要信息
- [ ] 不恶性竞争：不贬低竞争对手
- [ ] 不泄露机密：不泄露客户商业信息

**底线话术**：
```
"张总，我理解您的要求。但我需要诚实告诉您，
这个价格已经突破公司底线了，我确实没有权限再降。
不过，咱们可以想想其他办法..."
```
